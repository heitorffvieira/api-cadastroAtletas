# API do Projeto Cadastro de Atletas do Falcões Running.
Projeto pequeno, desenvolvido apenas para ter controle do número de camisas que serão encomendadas e quem serão os atletas com seus respectivos tamanhos e idade.

## Tecnologias Utilizadas:
- Java
- Spring Boot
- Spring JPA
- H2 DataBase
- DevTools
- Lombok
- Spring Web
